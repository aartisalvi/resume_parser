Project Overview:
The AI-Based Resume Parser is a machine learning application that automatically extracts and structures key information from resumes in PDF and DOCX formats.
It uses Natural Language Processing (NLP) techniques to convert unstructured resume data into structured fields for easy analysis and storage.

Objectives:
Automate resume screening
Reduce manual effort in candidate shortlisting
Improve efficiency of recruitment processes.

Technologies Used:
Python
NLP (NLTK)
Machine Learning
Pandas
Regular Expressions 
PDF & DOCX Parsing Libraries.

Testing Scope:
Tested with multiple resume formats
Validated accuracy of extracted fields
Handled missing or incomplete resume data.

Key Features:
Supports PDF and DOCX resume formats
Extracts:
Candidate Name
Email ID
Phone Number
Skills
Education
Work Experience
Text cleaning and preprocessing using NLP

Outputs structured data in JSON / CSV format

Can be integrated with ATS systems
